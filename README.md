# 🤖 Hope Assistant – Chatbot étudiant avec Groq API

**Version : 1.0.0** | Licence MIT

Petite application web Flask qui utilise l'API Groq (modèle **Llama 3.1 8B**) pour répondre aux questions des étudiants.  
Hope Assistant est un assistant intelligent, rapide et gratuit (dans la limite du quota Groq).

## ✨ Fonctionnalités
- Interface web simple (HTML/CSS)
- Dialogue en temps réel via requêtes AJAX
- Personnalité système : "assistant étudiant intelligent"
- Ouvre automatiquement le navigateur au démarrage
